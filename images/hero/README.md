# Foto Hero (natural, tanpa bingkai)

Simpan file:

```
images/hero/main.jpg
```

## Style yang cocok (mirip FDC)
- Close-up **senyum** / wajah ramah, atau dokter + pasien
- Background foto sebaiknya **terang / soft** biar nyatu ke mint wash
- **Tanpa bingkai** di UI — tepi foto di-blend otomatis ke background
- Hindari foto gelap atau crop keras

## Teknis
- Format: JPG / WebP
- Prefer landscape atau portrait soft, resolusi min ~1400px lebar
- Subject sedikit ke **kanan** (sisi foto) biar kiri bersih untuk teks

Refresh browser setelah file ditaruh.
